GR Logo

This is Geoff Ritter's personal logo containing a squared G with rounded corners and an intertwined reversed R on a square background with rounded corners. At the bottom is a bar containing, in all capital letters, the words GEOFF RITTER. This WebComponent aims to implement the full logo styleguide and can be used anywhere you would want a high resolution svg logo on the web. If instead, you are looking for the main variants of the logo in pure svg, then check the src folder for this project.
